# schema 验证


## 使用 Joi
### 安装包
```shell
# Node >= v12
$ npm install --save @hapi/joi
$ npm install --save-dev @types/hapi__joi
```


## 使用内置的ValidationPipe
ValidationPipe 从 @nestjs/common 包导入。结合 class-validator 和 class-transformer
```shell
npm i --save class-validator class-transformer
```
