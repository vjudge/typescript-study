# [jest](https://github.com/facebook/jest)


### 安装
```shell
$ yarn global add jest
# 或者
$ npm install -g jest
```


### 配置文件
```shell
# 创建一个基本的配置文件
$ jest --init
```


### typescript 项目
```shell
npm i --save-dev jest ts-jest @types/jest
# 或者
yarn add --dev jest ts-jest @types/jest
```
